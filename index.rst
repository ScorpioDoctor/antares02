欢迎学习 PyTorch 教程
============================

要了解如何使用PyTorch，请从我们的入门教程开始。 这个 :doc:`60分钟闪电战 </beginner/deep_learning_60min_blitz>`
是最常见的起点，它为如何使用PyTorch从基础到构建深层神经网络提供了一个广阔的视角。

一些考虑因素:

* 如果您想通过 IPython / Jupyter 交互地完成教程，那么每个教程都有一个Jupyter Notebook和Python源代码的下载链接。
* 在 `PyTorch Examples <https://github.com/pytorch/examples/>`_ 中，还提供了其他高质量的示例，
  包括图像分类、无监督学习、强化学习、机器翻译和许多其他应用。
* 您可以在 `PyTorch Docs <https://pytorch.org/docs>`_  中或通过内联帮助找到PyTorch API和layers的参考文档。
* 如果您希望改进教程部分，请在 `这里 <https://github.com/pytorch/tutorials>`_ 打开一个GitHub问题，提交您的反馈。
* 请查看我们的 `PyTorch Cheat Sheet <https://pytorch.org/tutorials/beginner/ptcheat.html>`_  ，以获得更多有用的信息。
* 最后，这里有一个指向PyTorch发行说明的链接 `PyTorch Release Notes <https://github.com/pytorch/pytorch/releases>`_ 。
